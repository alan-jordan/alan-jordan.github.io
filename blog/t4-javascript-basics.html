Javascript Basics

18th March 2017

How does JavaScript compare to HTML and CSS?

HTML and CSS are quite static. They define how a page is structured and how a page is styled. Javascript though is a fully fledged programming languages and allows us to do things dynamically. It can do math, check for the state of things, perform actions based on other conditions, search for things in data structures and a whole lot more.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

if (wakeUpState === 'tired') { goMakeCoffee(); } else if (wakeUpState === 'notTired') { goMakeTea(); }

So basically, I check to see if I'm tired when I wake up. If I am, then I run the goMakeCoffee function, in which I will make myself a cup of coffee. This function will contain all the steps required to make a cup of coffee. If I am not tired, then I will run the goMakeTea function instead, in which I will perform all the steps required to make a cup of tea.

Explain the difference between accessing data from arrays and objects.

Arrays are pieces of data stored in a clump sequentially. Accessing the data is a matter of going through each part of the array one by one. Eg myArray[0] then myArray[1]. Objcets however are a key value store and allow useful 'tagging' of that data. For example myObject[address] and myObject[name]. This allows the data to be much more easily understandable by a human.

Explain what functions are and why they are useful.

Functions enable programmers to re-use bits of code. They are great time savers and allow you to adhere to the "Don't repeat yourself" maxim.
