HTML, CSS & the DOM Part 2

2nd March 2017

What is grid based design?

Grid based design means breaking up sections of a web page into grids and positioning content appropriately. This particuarly suits the web as all elements on a page are boxes and the box model defines how elements are positioned and represented.

What is all the hype about responsive webpages?

How we view the web is changing. Desktops and laptops are getting higher resolution displays, mobile phones are now one of the most common ways of viewing web sites and tablets are also commonplace.

Because of this, designing a page for a 1024 x 768 monitor are over. Pages are loaded on numerous types of browsers with vastly varying screen sizes. Not taking this into account would mean that pages just won't look right on some devices. Practicing responsive design means that sites can dynamically adjust how they look based on screen size or device type.

What is semantic structure?

Semantic structure allows humans to read HTML code as well as browsers. For example, H1 is a semantic tag representing header 1. This is interpreted by the browser as it will know what it means. It is also easy for a human to understand. The same goes with the tag used to define images, IMG.
