Javascript Language

29th March 2017

In the JavaScript language, describe all the use cases for each of:

  • Parentheses () - Parentheses are used to tell the program to run an extra bit of the program called a function. You can pass extra bits of information into the function by placing them in between these parentheses. For example, you could call a function like this sayHello('Bob'). The function would then be able to do something like say Hello Bob!
  • You can also uses parentheses when creating a part of a program which you will need to loop over. For example, you could use what is called a for loop to run over a list of 'stuff' and then print it all out.
  • Brackets [] - Brackets are used to create something in Javascript called an Array. An array acts liks a list of 'stuff'. For example, you could use an array to store a list of vegetables for example. It would look like this var vegetables = ['peas','carrots','cabbage'];
  • Brackets are also used to access the contents of either arrays, or a slightly more specialised data structure called an object. For example, if you wanted to check what is the third vegetable in our array you could write console.log(vegetables[2]);
  • Braces {} - Braces are used to define an object. This is a bit like an array but rather than just being a collection of stuff which can be access using the number of the place it sits at in the array, it actually has some useful information. For example, you could create an object like this var Dog = {name: "Fido", legs: 4};
  • Braces are also used to indicate where parts of a loop end. For example, if you were looping through an array, your code would look something like this - for(i = 0; i < 10; i++) { console.log(foo[i]); } This would would loop through all 10 parts of the foo array and print out its contents.
  • Single quotes '' - Single quotes can be used when looking inside an object. For example, console.log(foo['bar']); would print out the contents of the bar property inside the foo object.
  • Double quotes "" - Double quoute scan also be used when looking inside an object as per the previous example. They are also used when defining something called a string. A string is a collection of characters. For example var testString = "I am a string!" is a string.
