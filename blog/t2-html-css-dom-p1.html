What are HTML, CSS and the DOM
  • HTML defines the structure of a web page
  • CSS defines how a web page looks
  • The DOM describers the underlying structure of a web site in a tree like structure

What is meant by boxifying design?

Boxifying design means breaking up a design into all of it's box like elements. This could be done using a graphics application and would allow the tranlsation of a design into something that can be carved up to be then rendered using HTML & CSS. This works because all elements in HTML are in fact boxes.

What is the box model?

The box model defines how an element on a site is displayed. The box comprises of:

  • Content: This is the actual content of the element. It could be an image, some text or other elements defined by the HTML.
  • Padding: This provides space around the Content before getting to any border.
  • Border: This allows developers to render a border around the element and the padding.
  • Margin: This provides space outside the border which other elements are not able to cross into.
